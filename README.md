# Flint

[![codecov](https://codecov.io/gh/TimelessOS/Flint/graph/badge.svg?token=IwZN4vN6WK)](https://codecov.io/gh/TimelessOS/Flint)

Flint is the universal package manager, designed for allowing multitudes of repositories and user only packaging.

## What about other tools?

Brew is good, but it's starkly not designed for linux.

Nix is also good, but it's not single-user or library focused.

This project aims to fix the downfalls of both, while still being compatible as an alternative.
